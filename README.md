# 📈 스톡인사이트 (StockInsight)
> **"주식이 처음인가요? 스톡인사이트와 함께라면 시장 흐름이 한눈에 보입니다."**

스톡인사이트는 복잡한 HTS/MTS 대신, 핵심 정보만을 모아 직관적으로 보여주는 **초보자 맞춤형 주식 현황판**입니다. 구글 파이낸스의 실제 데이터를 기반으로 시장 지수와 주요 종목의 움직임을 실시간으로 시뮬레이션합니다.

---

## 🌟 주요 특징

1. **직관적인 시장 지표**: 코스피, 나스닥 등 국내외 주요 지수를 한곳에서 확인하세요.
2. **실시간 데이터 시뮬레이션**: 주가 변동을 실시간으로 시뮬레이션하여 실제 시장의 생동감을 느낄 수 있습니다.
3. **나만의 관심 종목**: 평소 눈여겨보던 종목을 저장하고 언제든 다시 확인하세요.
4. **시각적 차트**: Chart.js를 활용한 깔끔한 차트로 가격 흐름을 쉽게 파악할 수 있습니다.
5. **눈이 편안한 다크 모드**: 신뢰감 있는 다크 테마로 오랜 시간 봐도 눈이 피로하지 않습니다.

---

## 📚 초보자를 위한 주식 용어 가이드

주식이 처음이신가요? 화면에 나오는 용어들을 쉽게 설명해 드립니다.

*   **KOSPI (코스피)**: 대한민국을 대표하는 주식 시장 지수입니다. 삼성전자, 현대차 같은 큰 기업들이 모여 있습니다.
*   **NASDAQ (나스닥)**: 미국의 기술주(애플, 테슬라 등) 중심 시장입니다. 전 세계 기술 트렌드를 보여줍니다.
*   **현재가**: 지금 이 순간 거래되고 있는 주식의 가격입니다.
*   **전일대비 (변동)**: 어제 장이 끝날 때 가격에 비해 얼마나 오르거나 내렸는지를 나타냅니다.
    *   🔴 **빨간색**: 어제보다 가격이 올랐음 (상승)
    *   🔵 **파란색**: 어제보다 가격이 내렸음 (하락)
*   **거래량**: 오늘 하루 동안 사고팔린 주식의 총 개수입니다. 거래량이 많을수록 시장의 관심이 높다는 뜻입니다.

## 🚀 시작하기

별도의 설치가 필요 없습니다! 

1. 이 저장소를 다운로드하거나 클론합니다.
2. `stock-dashboard` 폴더 내의 `index.html` 파일을 웹 브라우저(Chrome, Edge 등)로 실행하세요.
3. 바로 실시간 주식 현황판이 나타납니다.

---

## 🛠 사용된 기술 (Tech Stack)

*   **Frontend**: HTML5, CSS3 (Vanilla CSS), JavaScript (ES6+)
*   **Visualization**: [Chart.js](https://www.chartjs.org/) (데이터 시각화)
*   **Layout**: CSS Grid & Flexbox (반응형 디자인)

---

## 💡 주요 기능 사용법

1. **대시보드**: 전체적인 시장 지수와 주요 뉴스를 확인합니다.
2. **종목 검색**: 상단 검색창에 '삼성전자' 또는 종목 코드 '005930'을 입력해 보세요.
3. **상세 정보**: 종목을 클릭하면 자세한 가격 변동 차트와 통계를 볼 수 있습니다.
4. **관심 종목 등록**: 상세 화면에서 '+ 관심 종목 추가' 버튼을 눌러 나만의 리스트를 만드세요.
